# CPUセリフ・アルゴリズム強化メモ

## 反映内容

- CPUキャラクター定義に、指定仕様どおり `gender` / `style` / `catchphrase` / `motto` を整理して追加。
- かももどき、ワクもどき、リクもどきごとに、以下の場面別セリフを大幅追加。
  - ゲーム開始
  - 3枚パス
  - 開始時ペア浄化
  - リード時
  - フォロー時
  - フォロー不能時
  - 危険札処理時
  - トリック結果
  - ピック候補選択
  - 公開ピック
  - ピック結果
  - ババブタ公開
  - ペア浄化
  - シュート・ザ・ピッグ接近/発動
  - ラウンド終了
- `「プレイヤー名」さん` の形で、人間プレイヤーやピック相手の名前を参照して発言するように変更。
- 選択ルールに応じて、セリフ内で以下を参照するように変更。
  - 失点方式：`flat3` / `faceValue` / `spadeSuit`
  - ピック対象数
  - ババブタ失点タイミング
  - シュート・ザ・ピッグON/OFF
  - 終盤の手札枚数
  - CPU自身の手札リスク
- 人間プレイヤーが勝者/最弱になった場合でも、CPUが観戦コメントで場を盛り上げる `cpuTableTalk()` を追加。
- CPUピックの公平性は維持。ピック選択自体は裏向き候補から一様ランダムのまま。

## アルゴリズム面

既存のCPU評価関数を維持しつつ、セリフ判断と連動する情報を強化。

- かももどき：攻撃的。勝ちや危険札処理に積極的。毒舌・♡・ウホッあり。
- ワクもどき：大胆。高リスク高リターン、シュート志向、ランダム性強め。
- リクもどき：堅実。危険札、どろんこ失点、手札リスクを管理。
