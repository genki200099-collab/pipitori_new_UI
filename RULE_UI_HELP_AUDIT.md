# ルール選択UI追加改善メモ

## 追加内容

- ルール変更時の「試合傾向」自動説明を追加。
  - ラウンド数、失点方式、マッド・ピッグ、シュート・ザ・ピッグ、ババブタ失点、ピック対象数、開始時追加ルールを組み合わせて説明します。
- 各ルール項目に `?` ボタンを追加。
  - ラウンド数
  - 残り手札の失点
  - マッド・ピッグ
  - ババブタ失点
  - ババブタ失点タイミング
  - シュート・ザ・ピッグ
  - ピック対象数
  - 開始時3枚パス
  - 開始時ペア捨て
- `?` ボタン押下時に、個別ヘルプモーダルを表示。
- マッド・ピッグをOFFにした場合、シュート・ザ・ピッグを自動OFF・選択不可にするUI説明も更新。

## 確認

- `public/index.html` 内JavaScript構文チェック済み。
- `server.js` 構文チェック済み。
